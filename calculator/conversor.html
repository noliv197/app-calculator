Unit Conversor

Ferramenta para converter uma unidade em outras grandeza da mesma unidade. Para realizar a conversão basta identificar o tipo de unidade (Volume/Massa/Matéria/Temperatura) selecionar a unidade a ser transformada no campo a esquerda e a unidade alvo no campo a direita.
Para as conversões de Volume, Massa e Matéria o Calculo é feito usando as unidades de grandeza, por exemplo: de mL para L ocorre uma multiplicação por 1000.
Já para as conversoes de temperatura são utilizadas as formulas


\(F = 1.8 C + 32\) e \(K = C + 273.15\)
